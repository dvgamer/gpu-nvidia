## gpuminer-line api notification
gpuminer-line support Nvidia GTX1080 TI and `nicehash.com` only

### Environment
- `NICEHASH_ID` 
- `NICEHASH_KEY`
- `NICEHASH_WALLET`

```bash
npm i
npm run miner1
```
