## gpuminer-line api notification
gpuminer-line support `NVIDIA GTX1080 TI` and `nicehash.com` only 

![gpuminer-line](https://raw.githubusercontent.com/dvgamer/gpu-watcher/master/images/sample-v2.png)

- `NICEHASH_ID` 
- `NICEHASH_KEY`
- `NICEHASH_WALLET`
- `SLACK_URL`
- `LINE_TOKEN`
- `LINE_ID`
